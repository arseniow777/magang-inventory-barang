## Latar Belakang

Proyek ini dibuat untuk mengelola inventaris barang lintas lokasi secara terpusat.

## Tujuan Utama

- Menyediakan pencatatan barang dan unit barang yang konsisten.
- Mendukung alur permintaan, peminjaman, transfer, dan pelaporan barang.
- Menyediakan jejak audit agar aktivitas penting dapat dilacak.
- Mempercepat operasional tim melalui dashboard dan notifikasi.

## Ruang Lingkup

- Backend REST API berbasis Express dan Prisma.
- Frontend React berbasis fitur untuk modul bisnis inventory.
- Integrasi role-based access dan kontrak API terpadu.

## Non-Goal

- Tidak membahas konfigurasi infrastruktur cloud secara detail.
- Tidak membahas integrasi pihak ketiga di luar stack saat ini.
