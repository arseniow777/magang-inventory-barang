## Istilah

- `Item Master`: Data induk barang (nama, model, kategori, tahun).
- `Item Unit`: Unit fisik dari item master dengan kode unik.
- `Request`: Permintaan perpindahan/peminjaman/operasi barang.
- `Official Report`: Dokumen laporan resmi terkait aktivitas barang.
- `PIC`: Person in Charge untuk pengelolaan barang pada area tertentu.
- `Audit Log`: Jejak aktivitas penting sistem untuk keperluan audit.
