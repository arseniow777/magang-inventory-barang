## Kapan Harus Update Docs

- Menambah endpoint baru di backend.
- Mengubah kontrak request/response.
- Menambah fitur baru di frontend.
- Mengubah aturan role, izin, atau alur bisnis.

## Aturan Penulisan

- Gunakan bahasa Indonesia yang jelas dan ringkas.
- Tambahkan contoh bila ada format data khusus.
- Hindari informasi yang tidak terverifikasi dari source code.

## Alur Kontribusi

1. Update file `.md` atau `.mdx` sesuai section.
2. Jalankan docs lokal untuk preview perubahan.
3. Pastikan sidebar tetap rapi dan urutan halaman logis.
4. Tambahkan catatan pada `reference/changelog.md`.
